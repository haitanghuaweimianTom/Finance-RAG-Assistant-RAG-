金融研报问答助手 - 使用说明

配置 Key：右键点击 .env 文件，用记事本打开，把你的硅基流动 API Key 贴进去。
准备资料：把你想分析的 PDF 研报全部放进 data-rawpdf 文件夹。
首次运行：
双击 1.安装环境.bat (仅需运行一次，等待完成)。
双击 3.更新数据库.bat (它会处理你的 PDF 资料)。
双击 4.清空数据库.bat 则会清空之前你上传的pdf资料的处理数据，请删除data-rawpdf文件夹中不需要的pdf，添加你需要的pdf，重新双击3.更新数据库.bat 。
开始提问：双击 2.启动助手.bat，程序会自动弹出浏览器窗口。
注意：

如果电脑从未安装过 Python，运行第一个脚本时会自动弹出安装窗口，请勾选 "Add Python to PATH"。

运行期间请保持网络通畅。
